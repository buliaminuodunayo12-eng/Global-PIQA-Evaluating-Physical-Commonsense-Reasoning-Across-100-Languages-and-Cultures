ABOUT GLOBAL PIQA
Global PIQA is a large, community-built benchmark designed to evaluate physical and cultural commonsense reasoning across 100 language and cultures.
The project involved 335+ researchers from 65 countries, all contributing culturally specific questions and answers in their native languages.
